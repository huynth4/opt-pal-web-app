## Project Name

**OPT Pal** — a personalized job search app designed to connect international graduates with current H1B visa sponsors.

## Category

Web Development (Flask)

## Description

OPT Pal is a Flask web app that lets users **filter and browse** **H-1B** employer statistics derived from **public USCIS data** (the same family of information published through the [USCIS H-1B Employer Data Hub](https://www.uscis.gov/h-1b/h-1b-employer-data-hub)). Users can discover suitable work prospects by filtering down by different categories available in the dataset (for example from **industry** to **state** to **city**) and see meaningful insights from a autogenerated graph/chart. The app is aimed at **international students and recent graduates** who want a faster way to explore potential organizations that sponsor H1B visas than downloading and pivoting spreadsheets. A short **data and limitations** section explains what the numbers mean and that they are **not** legal advice or a guarantee of future sponsorship.

## Features

List the main features you plan to build. Start with the essentials (your MVP) and then list stretch goals.

**Must have (MVP):**

- [ ] **Quiz Onboarding:** starts the personalization process for the user by asking them to choose the state(s) they want to discover work opportunities, then the specific cities, then industries, and the size of the company to which they are aiming for. This could be displayed in a dropdown form or a multi-select form.
- [ ] **Dashboard View:** summarizes filtered results with quick statistics, such as top employers with most approvals and number of approvals by state. It also visualizes the data using simple bar charts, so users could understand the patterns better.
- [ ] **All filters compose:** every step narrows the same SQL query in `search.py`
- [ ] **Provenance and disclaimer:** USCIS source, fiscal year(s), and a plain-language note that historical data doesn't predict individual outcomes


**Nice to have (stretch):**

- [ ] **Export:** download the current filtered table as CSV.
- [ ] **Compare:** side-by-side snapshot for two states or two employers (same metrics).
- [ ] **Visualize:** add a globe-based view using Three.js or Mapbox GL JS to display employer cities as markers and draw arcs from employer locations to a “you are here” point chosen by the user.

## Tech Stack

- **Python 3**
- **Flask** — routes, forms/query params for filters, Jinja2 templates
- **Pandas** — one-time or scripted cleaning of USCIS exports into a tidy CSV/Parquet checked into the repo (or loaded at startup)
- **HTML / CSS** — layout for table and filters; optional **Chart.js** (or similar) only if you add stretch charts
- **Data storage (MVP):** static file(s) in the repo (e.g., CSV); optional **SQLite** later if the dataset grows and you want faster queries
